# Kesha Platform

Telegram bot platform for real estate brokers — EGRN requests, CRM integration, listing parser, watermark removal, team analytics.

## Stack
Python · Docker · PostgreSQL · Linux VPS

## Features
- EGRN property registry requests
- Listing parser with anti-bot bypass (Cian.ru)
- CRM integration (lot and contact creation)
- Watermark removal from property photos
- Team analytics dashboard
